# Proyecto Medoro 3 – Optimización de Tiempos de Preparación y Producción (2025)

Este archivo PBIX está preparado para ser usado sin conexión a SQL Server.

✅ Base de datos utilizada: Sheet1.xlsx (incluido en la carpeta).  
✅ No requiere acceso al servidor ni a las vistas SQL originales.

## Instrucciones para abrir correctamente:
1. Abrir el archivo `Medoro4_Jose.pbix` en Power BI Desktop.
2. Asegurarse de que el archivo `Sheet1.xlsx` esté en la misma carpeta que el PBIX.
3. El PBIX está configurado en modo Importación, no es necesario actualizar conexiones.
4. Los filtros por ID y Fecha ya están activos y aplican a todos los gráficos.
5. No tocar configuraciones de relaciones o fuentes de datos.

## Funcionalidades disponibles:
- Dashboard de análisis de eficiencia de preparación vs producción.
- Indicadores de tiempo de preparación (%) y semáforo de eficiencia.
- Gráfico de barras de eventos de producción.
- Gráfico de dispersión entre horas de preparación y horas de producción.
- Tabla detallada con cálculo de % de tiempo perdido en preparación.

IMPORTANTE: No eliminar ni mover el archivo Excel `Sheet1.xlsx` para garantizar que el PBIX funcione correctamente.
